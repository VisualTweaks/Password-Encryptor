# Passsword-Encryptor

This password encryptor was created using Vanilla JavaScript.


## Usage/Examples

```javascript
var password = makePassword();

encrypted = encryptpassword(password);

decrypted = decryptpassword(encrypted);
```


## Features

- Encrypting Passwords
- Decrypting Passwords
- Creating Passwords
